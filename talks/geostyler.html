New Features

GeoStyler Context

Global configuration of GeoStyler 

                const myContext = {
                  "composition": {
                    "LineEditor": {
                      "opacityField": {
                        "visibility": false
                      },
                      "widthField": {
                        "default": 5
                      }
                    },
                    "IconEditor": {
                      "visibility": false
                    }
                  },
                  "locale": {
                    "Style": {
                      "nameFieldLabel": "Style Name"
                    }
                  }
                };

                <GeoStylerContext.Provider value={myContext}>
                  <Style style={style} compact={true} />
                </GeoStylerContext.Provider>
